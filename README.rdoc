== README
A Ruby on Rails applicaiton for url shortening.

== Note:
- Using sqlite3 - install it before you run this application.

== Installation 
- Clone repo: git clone git@github.com:krishnau2/url_shortner.git
- cd url_shortner
- bundle install
- rake db:create
- rake db:migrate
- rails server
- Go to http://localhost:3000
